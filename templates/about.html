About This Project

This Gold Price Prediction web application utilizes the Facebook Prophet forecasting model â€” a powerful open-source tool developed by Meta for time series analysis and prediction. Prophet is well-known for handling missing data, outliers, and seasonality with ease, making it an excellent choice for financial forecasting.

The dataset used for training the model spans a rich historical range from December 29, 1978 to April 29, 2024. This extensive dataset allows the model to capture long-term patterns, trends, and seasonality in gold prices.

The predictions generated here are meant for short-term insights, making them especially helpful for individual traders, students, and financial enthusiasts looking to analyze gold market trends.

The application is powered by Flask on the backend and styled with a professional, minimalist aesthetic using custom CSS. The goal is to keep the user experience smooth, informative, and visually appealing.

All visualizations and trends are updated using the most recent available data. We aim to continue improving the system with more features such as exportable reports, model comparison tools, and real-time data updates.

This web application was developed solely for academic purposes as part of a final year project in the field of machine learning and data analytics. It is not intended for commercial use or public deployment. The system showcases the implementation of forecasting techniques using historical data and aims to demonstrate the practical application of data science methodologies in a real-world scenario. While the predictions and features are functional, the web app is designed for educational demonstration only and will not be hosted or maintained beyond the scope of this project.
